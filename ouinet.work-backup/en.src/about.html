How CENO Browser Works

Most web traffic in countries that filter internet is routed through centralized exchanges, such as Google.

This design gives the censor an advantage in locating and blocking popular proxies, VPN servers, relays, etc;

In these cases, reaching origin web servers is difficult or impossible, and your device will not be able to get that content.

CENO is a mobile web browser for storing and sharing web content anywhere.

CENO users create small-world networks to connect to each other, and thereafter to a bridge outside the censored zone.

Once a website is accessed by a single CENO user, it is stored and shared inside the country peer-to-peer.
